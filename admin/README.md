# MrMaper Blog Admin

یک پنل مدیریت محلی با ورود (نام کاربری/رمز عبور) برای نوشتن و انتشار پست‌های وبلاگ
روی سایت استاتیک GitHub Pages. توکن گیت‌هاب فقط روی سرور نگه داشته می‌شود و هرگز
به مرورگر فرستاده نمی‌شود.

## راه‌اندازی

1. نصب وابستگی‌ها:
   ```
   cd admin
   npm install
   ```
2. ساخت فایل `.env` در پوشه `admin` با محتوای:
   ```
   GITHUB_TOKEN=ghp_توکن_شما
   ADMIN_USER=admin
   ADMIN_PASS=یک_رمز_قوی
   ```
   (توکن باید دسترسی `repo` داشته باشد — از GitHub → Settings → Developer settings → PAT)
3. اجرا:
   ```
   node server.js
   ```
4. مرورگر را باز کنید: http://localhost:3000 و با نام کاربری/رمز عبور وارد شوید.

## استفاده

- فرم شامل عنوان انگلیسی/فارسی، تاریخ، نامک (slug)، برچسب‌ها و متن مارک‌داون
  (جداگانه برای هر دو زبان) است.
- لیست پست‌های موجود برای ویرایش در دسترس است.
- با زدن «ذخیره و انتشار»، فایل در `_posts/` کامیت می‌شود و سایت پس از ~۱ دقیقه
  به‌روز می‌شود.

## میزبانی (اختیاری)

این سرور را می‌توانید روی یک VPS یا سرویس مثل Render/Fly اجرا کنید تا از هر
جایی با ورود به آن پست منتشر کنید. متغیرهای محیطی `GITHUB_TOKEN`، `ADMIN_USER`،
`ADMIN_PASS` را در محیط میزبان تنظیم کنید.
